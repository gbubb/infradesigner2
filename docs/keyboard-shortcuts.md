# Keyboard Shortcuts

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + B` | Toggle Sidebar | Shows/hides the component library sidebar |

## Form Interactions

| Context | Key | Action |
|---------|-----|--------|
| Connection Rule Form | `Enter` | Add new criteria item (when input field is focused) |

## Navigation

| Context | Key | Action |
|---------|-----|--------|
| Carousel Components | `Arrow Left` | Navigate to previous item |
| Carousel Components | `Arrow Right` | Navigate to next item |

## Notes

- **Mac Users**: Use `Cmd` key instead of `Ctrl`
- **Windows/Linux Users**: Use `Ctrl` key
- All shortcuts are case-insensitive
